# 🌈 SmartKids 365 - Nền tảng giáo dục mầm non

Ung dụng quản lý giáo dục mầm non hiện đại với công nghệ AI, kết nối nhà trường - giáo viên - phụ huynh.

## 🚀 Các tính năng chính

### 📋 Điểm danh thông minh
- QR code scanning
- Nhận diện khuôn mặt
- Thống kê realtime

### 🤖 AI Giáo án
- Soạn giáo án tự động
- Tạo trò chơi và hoạt động
- Nhận xét trẻ bằng AI

### 💌 Kết nối phụ huynh
- Chat trực tiếp
- Nhật ký lớp
- Thông báo tức thì

### 📈 Theo dõi phát triển
- Biểu đồ tăng trưởng
- Đánh giá kỹ năng
- Báo cáo tiến bộ

## 🛠️ Công nghệ sử dụng

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## 📦 Cài đặt

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm run dev

# Build production
npm run build

# Preview production build
npm run preview
```
